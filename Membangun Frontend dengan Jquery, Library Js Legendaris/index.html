1Ô∏è‚É£ Pengenalan jQuery

Apa itu jQuery?

jQuery adalah library JavaScript yang mempermudah manipulasi DOM, handling events, dan AJAX. Syntax-nya lebih singkat dibanding vanilla JavaScript.

// Cara lama (Vanilla JS)
document.getElementById('myId').style.color = 'red';

// Cara jQuery (lebih singkat)
$('#myId').css('color', 'red');

2Ô∏è‚É£ jQuery Selectors

a) Selector Dasar

Syntax: $(selector) - Memilih elemen HTML

// ID Selector (memilih elemen dengan id tertentu)
$('#myId')

// Class Selector (memilih elemen dengan class tertentu)
$('.myClass')

// Element Selector (memilih semua elemen tertentu)
$('p')

// Attribute Selector (memilih berdasarkan atribut)
$('[name="username"]')

Ini adalah paragraph yang akan berubah warna

b) Selector Kombinasi

// Descendant Selector
$('div p')  // Semua p di dalam div

// Child Selector
$('div > p')  // p yang merupakan child langsung dari div

// Multiple Selector
$('h1, p, span')  // Semua h1, p, dan span

// Filter
$('p:first')  // p pertama
$('p:last')   // p terakhir
$('p:even')   // p dengan index genap
$('p:odd')    // p dengan index ganjil

3Ô∏è‚É£ DOM Manipulation (Memanipulasi Elemen)

a) Mengubah Konten

// html() - Mengubah/mengambil HTML
$('#myId').html('<b>Konten Baru</b>');

// text() - Mengubah/mengambil text
$('#myId').text('Hanya Text');

// val() - Mengubah/mengambil value input
$('#input').val('Nilai Baru');
Konten Asli

b) Mengubah Atribut

// attr() - Mengubah/mengambil atribut
$('img').attr('src', 'gambar.jpg');
$('a').attr('href', 'https://google.com');

// removeAttr() - Menghapus atribut
$('input').removeAttr('disabled');

c) Mengubah CSS

// css() - Mengubah style CSS
$('#myId').css('color', 'red');
$('#myId').css('background', 'yellow');

// Mengubah multiple properties
$('#myId').css({
    'color': 'red',
    'background': 'yellow',
    'font-size': '20px'
});

// addClass() - Menambah class
$('#myId').addClass('active');

// removeClass() - Menghapus class
$('#myId').removeClass('active');

// toggleClass() - Toggle class
$('#myId').toggleClass('active');
Elemen Target

d) Membuat/Menghapus Elemen

// append() - Menambah elemen di akhir
$('#container').append('<p>Elemen Baru</p>');

// prepend() - Menambah elemen di awal
$('#container').prepend('<p>Elemen Baru</p>');

// after() - Menambah elemen setelah
$('#myId').after('<p>Setelah</p>');

// before() - Menambah elemen sebelum
$('#myId').before('<p>Sebelum</p>');

// remove() - Menghapus elemen
$('#myId').remove();

// empty() - Menghapus semua isi
$('#container').empty();

Elemen Awal

4Ô∏è‚É£ jQuery Events (Penanganan Kejadian)

a) Event Dasar

// click() - Klik
$('#btn').click(function() {
    console.log('Tombol diklik');
});

// on() - Cara modern & lebih fleksibel
$('#btn').on('click', function() {
    console.log('Tombol diklik');
});

// off() - Menghapus event
$('#btn').off('click');
Output: Siap di-klik

b) Event Form

// focus() - Saat input fokus
$('#input').focus(function() {
    console.log('Input fokus');
});

// blur() - Saat input blur (keluar fokus)
$('#input').blur(function() {
    console.log('Input blur');
});

// change() - Saat value berubah
$('#select').change(function() {
    var value = $(this).val();
});

// submit() - Saat form disubmit
$('form').submit(function(e) {
    e.preventDefault(); // Cegah default behavior
});
Output: Belum ada event

c) Mouse Events

// mouseenter() - Kursor masuk
$('#element').mouseenter(function() { });

// mouseleave() - Kursor keluar
$('#element').mouseleave(function() { });

// hover() - Gabungan mouseenter & mouseleave
$('#element').hover(
    function() { /* Masuk */ },
    function() { /* Keluar */ }
);

// mousemove() - Kursor bergerak
$('#element').mousemove(function(e) {
    console.log(e.pageX, e.pageY);
});
Hover di sini

7Ô∏è‚É£ AJAX (Asynchronous JavaScript and XML)

Mengambil Data Tanpa Refresh Halaman

// $.ajax() - Cara lengkap
$.ajax({
    url: 'https://api.github.com/users/github',
    type: 'GET',
    dataType: 'json',
    success: function(data) {
        console.log(data); // Data berhasil diambil
    },
    error: function(error) {
        console.log('Error:', error);
    }
});

// $.get() - Cara singkat GET
$.get('data.json', function(data) {
    console.log(data);
});

// $.post() - Cara singkat POST
$.post('submit.php', { name: 'John' }, function(data) {
    console.log(data);
});

// $.getJSON() - Khusus JSON
$.getJSON('data.json', function(data) {
    console.log(data);
});

8Ô∏è‚É£ jQuery Utility Functions

Fungsi Utilitas Penting

// $.each() - Loop
$.each([1, 2, 3], function(index, value) {
    console.log(index + ': ' + value);
});

// $.map() - Transform array
var doubled = $.map([1, 2, 3], function(val) {
    return val * 2; // [2, 4, 6]
});

// $.inArray() - Cari dalam array
var index = $.inArray(2, [1, 2, 3]); // 1

// $.extend() - Merge object
var obj = $.extend({a: 1}, {b: 2}); // {a: 1, b: 2}

// $.trim() - Hapus whitespace
$.trim('  hello  '); // 'hello'

// $.parseJSON() - Parse JSON string
var obj = $.parseJSON('{"name": "John"}');

// $.isArray() - Cek array
$.isArray([1, 2, 3]); // true

// $.isEmptyObject() - Cek object kosong
$.isEmptyObject({}); // true

üîü Best Practices & Tips

1. Cache jQuery Object

// ‚ùå Buruk - Memilih berkali-kali
$('#myId').css('color', 'red');
$('#myId').css('font-size', '20px');

// ‚úÖ Baik - Cache sekali saja
var $element = $('#myId');
$element.css('color', 'red');
$element.css('font-size', '20px');

2. Gunakan Class untuk Styling

// ‚ùå Buruk - Inline CSS
$('#myId').css({
    'color': 'red',
    'background': 'yellow',
    'padding': '10px'
});

// ‚úÖ Baik - Gunakan class
$('#myId').addClass('active-style');

/* CSS */
.active-style {
    color: red;
    background: yellow;
    padding: 10px;
}

3. Event Delegation

// ‚ùå Buruk - Bind setiap element
$('.btn').click(function() { });

// ‚úÖ Baik - Delegate ke parent (untuk dynamic elements)
$(document).on('click', '.btn', function() {
    console.log('Button diklik');
});

4. Gunakan Var/Let untuk Menyimpan Reference

// ‚úÖ Baik
var $myElement = $('#myId');
$myElement.addClass('active');
$myElement.text('Updated');

// ‚úÖ Lebih Modern (ES6)
const $myElement = $('#myId');
$myElement.addClass('active');

5. Handle Error dengan Benar

$.ajax({
    url: 'data.json',
    success: function(data) {
        console.log('Sukses:', data);
    },
    error: function(xhr, status, error) {
        console.log('Error:', error);
    },
    complete: function() {
        // Selalu dijalankan, baik sukses atau error
        console.log('Request selesai');
    }
});
